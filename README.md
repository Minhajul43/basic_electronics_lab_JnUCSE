# ⚡ Basic Electronics Lab - JnU CSE

A collection of 30+ LTspice-based Basic Electronics laboratory simulations covering transistor circuits, circuit analysis, waveform visualization, and practical electronics experiments.

## 📖 Overview

This repository contains LTspice simulation files developed for the Basic Electronics Lab course of Jagannath University (JnU) CSE.

The simulations help students understand electronic circuits through practical visualization and analysis instead of relying only on theoretical concepts.

## 🛠 Requirements

- LTspice
- Basic knowledge of Electronics

Download LTspice:

https://www.analog.com/en/design-center/design-tools-and-calculators/ltspice-simulator.html

## 📂 Repository Contents

The repository includes:

- LTspice schematic files (`.asc`)
- Simulation output files (`.raw`)
- Operating point analysis files (`.op.raw`)
- Log files (`.log`)

Example topics:

- Common Emitter Transistor
- Common Collector Transistor
- Common Base Transistor
- Amplifier Circuits
- DC Analysis
- AC Analysis
- Transistor Characteristics
- Electronic Circuit Simulations

## 🚀 Running the Simulations

1. Clone the repository

```bash
git clone https://github.com/Minhajul43/basic_electronics_lab_JnUCSE.git
```

2. Open LTspice

3. Open any `.asc` file

4. Click **Run** to simulate the circuit

5. Analyze the generated waveforms and results

## 🎯 Learning Outcomes

By exploring these simulations, you can:

- Understand transistor configurations
- Analyze electronic circuits
- Visualize voltage and current waveforms
- Learn practical electronics concepts
- Improve circuit design understanding
